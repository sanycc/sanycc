## Profile

**Master's student in Mechanical Engineering** at **Beijing University of Posts and Telecommunications**, with a research focus on **robot motion planning**. My work centers on robot system integration and development, manipulator motion planning, fast path planning, online trajectory planning, Qt-based human-machine interfaces, 3D simulation, and real-robot debugging.

I have hands-on engineering experience across algorithm design, software module development, system integration, on-site commissioning, and hardware validation. My project background includes robotics systems for **aviation industry applications**, **embodied-intelligence robotic pilot control**, and **additive manufacturing**. I care deeply about making robotics algorithms stable, debuggable, and usable on real robotic platforms.

I also make active use of **Claude Code**, **Codex**, and other AI development tools for code generation, refactoring, debugging assistance, documentation, and rapid prototype validation, integrating AI into my daily engineering workflow.

## Project Experience

### Dual-Arm Robot Development for Aviation Industry Applications

**Role: Software System Lead**

Led software system integration, development, and debugging for a dual-arm robotic platform, building core capabilities across motion planning, online trajectory updates, human-machine interaction, and function integration for complex aviation industry tasks.

- Integrated software modules, debugged system components, and supported full-system commissioning on a real dual-arm robot platform.
- Implemented and debugged linear planning, circular-arc planning, and joint-space planning algorithms to improve path controllability, trajectory continuity, and execution stability.
- Optimized path generation and trajectory update workflows around on-site debugging needs and complex operation constraints.
- Developed Qt-based robot debugging and HMI software, integrating 3D simulation, status monitoring, parameter configuration, and function-module debugging.
- Integrated remote operation and on-site interaction functions to support validation and rapid issue reproduction during complex tasks.
- Participated deeply in real-robot testing, exception diagnosis, parameter tuning, and system iteration.

**Outcome:** Built practical experience in deploying motion-planning algorithms, HMI software, and system-integration capabilities into a real dual-arm robotic system for industrial applications.

### Embodied-Intelligence Robotic Pilot Control Algorithm Research

**Role: Technical Lead**

Served as technical lead for the project, coordinating the technical scheme and development progress while focusing on manipulator motion-planning algorithms for operating aircraft cockpit control devices.

- Planned the technical roadmap and decomposed work across algorithm development, system integration, and validation.
- Researched motion-planning methods for manipulator operation in constrained aircraft cockpit environments.
- Designed and implemented planning algorithms around control-device position constraints, constrained-space collision avoidance, and stable execution.
- Drove closed-loop iteration from technical design and simulation validation to real-robot debugging, improving reliability for cockpit operation tasks.

**Outcome:** Gained technical ownership experience for embodied-intelligence tasks and translated high-level operation goals into executable and verifiable robot motion-planning workflows.

### Manipulator Motion-Planning Algorithm Library for Additive Manufacturing

**Role: Motion Planning Algorithm Developer**

Developed time-optimal joint-space trajectory planning algorithms and supported stable cross-platform deployment of a manipulator motion-planning algorithm library.

- Developed time-optimal joint-space trajectory planning algorithms to improve execution efficiency and motion continuity for additive manufacturing scenarios.
- Optimized the trajectory-planning workflow around path continuity, velocity constraints, joint constraints, and execution stability.
- Contributed to algorithm library interface design, functional validation, and platform adaptation to improve portability and engineering usability.
- Supported stable deployment across different platforms for efficient integration into downstream manipulator application systems.

**Outcome:** Built experience in engineering a motion-planning algorithm library from implementation and interface design to cross-platform adaptation and stable application.
